# Score Zone

**Score Zone** is a mobile application that displays fixtures and player statistics for popular football leagues. This app provides real-time data, including upcoming match fixtures, player stats, and league standings. 

---

## Features

- **Fixtures**: View upcoming and past match fixtures for popular football leagues.
- **League Table**: View the current league standings with team rankings, points, and goal statistics.
- **Team Info**: Get detailed information about teams, including squad members and their stats.
- **Top Scorers**: Keep track of the top goal scorers in each league.
- **Real-time Data**: Stay updated with the latest football data powered by reliable API.

---

## Technologies

- **Flutter**: The app is built using Flutter for a smooth and responsive cross-platform experience.
- **Provider Pattern**: Manages app state using the Provider pattern, making it scalable and easy to maintain.
- **API Integration**: Integrates with football data APIs to provide live fixture and player data.
- **Dio & Retrofit**: Handles HTTP requests and integrates with the backend API using Dio and Retrofit for efficient data fetching.
- **Json Annotation**: Used for JSON serialization and deserialization to map the API data into Flutter models.
- **Auto Route Package**: Simplifies routing and navigation within the app using the Auto Route package.

## API Reference

The data for fixtures, player statistics, and league standings is provided by [Football Data API](https://www.football-data.org). This API provides access to real-time football data from various leagues around the world.

For more information on how to use the API or to get your API key, please visit the website.
